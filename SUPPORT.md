# Support

Transit is a portfolio project and has no service-level agreement.

Use [GitHub Issues](https://github.com/mgkdante/transit/issues) for reproducible
software defects and focused feature proposals. Include the affected URL or
command, expected and observed behavior, reproduction steps, and relevant logs
with secrets removed.

Security reports belong in [SECURITY.md](SECURITY.md), never a public issue.

Transit is not an official STM service. The dashboard and public data are
informational and must not be used for emergencies or trip-critical decisions.
Use official transit channels for authoritative service information.
